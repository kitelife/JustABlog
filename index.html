JustABlog

a blog website for coding practice

Dependencies

  • Python Microframework --- Flask (include Werkzeug, Jinja 2)
  • Database Engine --- Python-SQLite3
  • Python-MarkDown

Setup

In command-line, do:

  • git clone git://github.com/youngsterxyf/JustABlog.git
  • python initDB.py
  • python app.py
  • google-chrome 127.0.0.1:5000
